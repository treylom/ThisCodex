# ⚠️ Codex plugin packaging — not the install path (yet)

This directory exists so ThisCodex follows the OpenAI plugin packaging
conventions (`openai/plugins` / `obra/superpowers` sync pattern), but plugin
discovery is **incomplete on current codex releases** — do not use it as your
entry path.

Install instead via:

```bash
npx github:treylom/ThisCodex init     # guided onboarding (recommended)
```

or skill placement (`skills/thiscodex/` → your Codex skills directory). See
`README.md` §Help & Getting Started and `skills/thiscodex/SKILL.md`.
